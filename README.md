The Goal
This project uses a KNN model to classify data based on anonymized features. The focus was on finding the most accurate K value to ensure the model isn't just guessing, but actually learning patterns.

What I Did
Standardization: Scaled the data so large numbers wouldn't bully the smaller ones during distance calculations.

The Elbow Method: Ran different versions of the model to track the error rate.

Optimization: Found that K=30 was the sweet spot for accuracy.
